# 🤖 Бот для презентаций — Инструкция по запуску

Следуй этим 5 шагам. Каждый занимает 2–5 минут.

---

## Шаг 1 — Создай бота в Telegram (2 мин)

1. Открой Telegram, найди бота **@BotFather**
2. Напиши ему: `/newbot`
3. Придумай название, например: `Презентационный бот`
4. Придумай username (должен заканчиваться на `bot`), например: `my_prez_bot`
5. BotFather пришлёт **токен** — строчку вида `7234567890:AAHxxx...`
6. Скопируй его, он понадобится на шаге 3

---

## Шаг 2 — Получи ключ Claude API (5 мин)

1. Открой **https://console.anthropic.com**
2. Зарегистрируйся или войди
3. Нажми **"API Keys"** → **"Create Key"**
4. Скопируй ключ (начинается с `sk-ant-...`)
5. Пополни баланс: **"Billing"** → минимум $5 (хватит на сотни презентаций)


---

## Шаг 3 — Загрузи код на GitHub (3 мин)

1. Открой **https://github.com** → зарегистрируйся если нет аккаунта
2. Нажми **"New repository"** → назови `prez-bot` → **"Create repository"**
3. Нажми **"uploading an existing file"**
4. Перетащи все файлы из папки `prez_bot` (которую ты получил)
5. Нажми **"Commit changes"**

---

## Шаг 4 — Задеплой на Railway (3 мин)

1. Открой **https://railway.app** → войди через GitHub
2. Нажми **"New Project"** → **"Deploy from GitHub repo"**
3. Выбери репозиторий `prez-bot`
4. После деплоя нажми на проект → **"Variables"**
5. Добавь переменные окружения (нажимай "Add variable"):

   | Имя переменной   | Значение                          |
   |------------------|-----------------------------------|
   | `TELEGRAM_TOKEN` | токен от BotFather (шаг 1)        |
   | `ANTHROPIC_API_KEY` | ключ от Anthropic (шаг 2)     |
   | `HUMAN_SPECIALIST` | `@username` твоего коллеги      |
   | `BRAND_FONT`     | `Calibri` (или другой шрифт)      |

6. Railway автоматически перезапустит бота

---

## Шаг 5 — Настрой корпоративные цвета (опционально)

Открой файл `config.py` и измени `BRAND_COLORS`:

```python
BRAND_COLORS = {
    "primary":   (0, 82, 147),    # Основной цвет заголовков (RGB)
    "secondary": (255, 255, 255), # Фон
    "accent":    (255, 163, 0),   # Акцентный цвет
    "text":      (33, 33, 33),    # Цвет текста
}
```

Значения RGB найди в брендбуке компании или попроси у дизайнера.

---

## ✅ Готово!

Открой Telegram, найди своего бота и напиши `/start`.

**Что умеет бот:**
- 👔 **Топ-менеджер** — перенаправляет к указанному специалисту
- 🏢 **Внутренний отчёт** — правит шрифты, цвета, выравнивание
- 🤝 **Для клиентов** — Claude переписывает контент итерациями
- 📝 **Рабочий материал** — быстрая шлифовка структуры

## ❓ Частые вопросы

**Бот не запускается?**
Проверь в Railway → Deployments → Logs. Скорее всего не заданы переменные окружения.

**Ошибка с API ключом?**
Убедись что пополнил баланс на console.anthropic.com → Billing.

**Хочу изменить поведение бота?**
Напиши мне — помогу настроить любую ветку логики.
